GTIFWriteKeys() - Write GeoTIFF Keys

    #include <geotiff.h>
    void GTIFWriteKeys(GTIF * gtif);
This function flushes all the GeoTIFF keys that have been set with the GTIFKeySet() function into the associated TIFF file. GTIFKeySet() should be called before GTIFFree() is used to deallocate a GeoTIFF access handle.

See Also: GTIFWriteKeys(), GTIFFree()
