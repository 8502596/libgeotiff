XTIFFClose() - Close GeoTIFF File

Usage:

    #include <xtiffio.h>
    void XTIFFClose( TIFF * tif );

Details:

The XTIFFClose() function is used to close a file opened with XTIFFOpen(). The file handle returned by XTIFFOpen() is passed to XTIFFClose() as tif.

If a GTIF structure was created with GTIFNew() for this file, it should be freed with GTIFFree() before calling XTIFFClose().
